Tempo Input

Tempo is expressed in BPM and can be set in two different ways :

  • Enter the value in the tempo input field then hit 'enter'

    • You can enter any numerical value between 1 and 999. If you enter a non-numerical value, or a value outside this range, BPM jumps back to the last valid value or to the default (60 BPM) if none has been set yet.

  • Use your mouse wheel while hovering the tempo input field

    • This will increase or descrease the value by integer steps and automatically recompute delays and LFOs

QREST Guide utilisateur

Bienvenue dans le guide utilisateur de Qrest !

Qrest est une boîte à outils qui vous permet de déterminer le tempo d'une compisition musicale et de calucler les temps de délai pour les durées de notes les plus courantes.

Voici la liste des fonctionnalités abordées dans ce guide :

  • Saisie du tempo
  • Tap Tempo
  • Choix des variantes
